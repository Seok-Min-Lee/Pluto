# 🎮 플레이어 조작 시스템 가이드 (Player Input Guide)

이 문서는 `Pluto` 프로젝트의 플레이어 조작 방식과 그에 따른 액션 피드백 정책을 정의합니다. 하데스(Hades)와 같은 빠르고 쾌적한 조작감을 목표로 합니다.

---

## 1. 기본 조작 체계 (Keyboard & Mouse)

| 액션 | 입력 (Default) | 설명 | 비고 |
| :--- | :--- | :--- | :--- |
| **이동 (Move)** | W, A, S, D | 캐릭터 8방향 이동. | 카메라 이동 없음 (고정 쿼터뷰) |
| **공격 (Attack)** | Mouse Left | 기본 무기 공격 (3단 콤보). | |
| **특수 공격 (Special)**| Q Key | 무기 고유 특수 기술 발동. | |
| **마법 (Magic)** | Mouse Right | 혈석 (Bloodstone)을 소모하여 투사체 발사. | 운용 방식(비설치/설치)에 따라 회수 로직 상이함. |
| **돌진 (Dash)** | Space Bar | 지정된 방향으로 짧고 빠르게 이동. | 이동 중 무적 프레임 존재 |
| **신의 원조 (Call)** | R Key | 신 게이지 소모하여 강력한 기술 발동. | |
| **상호작용 (Interact)**| E Key | 은혜 획득, NPC 대화, 문 열기. | |

## 2. 고급 조작 메커니즘 (Advanced Mechanics)

### A. 입력 선입력 (Input Buffering)
액션의 후딜레이가 끝나기 전(약 0.1~0.2초 전)에 입력된 다음 명령을 기억했다가, 동작 가능 시점에 즉시 실행합니다.
- **적용 대상:** 공격 -> 공격 (콤보), 공격 -> 대시, 대시 -> 공격.

### B. 대시 캔슬 (Dash Canceling)
공격 동작 중 언제든지 대시 입력을 통해 현재 동작을 캔슬하고 회피할 수 있습니다.
- **예외:** 강력한 한 방 공격이나 런지(Lunge) 구간 중 '캔슬 불가' 프레임이 존재할 수 있음.

### C. 마법 운용 및 혈석 회수 (Magic & Bloodstone)
은혜의 종류에 따라 혈석 자원의 순환 방식이 두 가지로 나뉩니다.

1.  **비설치형 (Target-Embedded):**
    *   **로직:** 적에게 투사체를 맞추면 혈석이 적의 몸에 박힙니다.
    *   **회수:** 적이 사망하거나 일정 시간(`ExitingTime`)이 지나면 혈석이 바닥에 떨어지며, 플레이어가 **직접 접근하여 회수**해야 다시 사용할 수 있습니다.
2.  **설치형 (Deployable / Area):**
    *   **로직:** 특정 위치에 수정탑을 소환하거나 가스 안개를 생성합니다.
    *   **회수:** 효과가 지속되는 동안 해당 혈석 슬롯은 비활성화(점유) 처리되며, 효과가 종료되면 플레이어에게 **자동으로 복귀(재충전)**됩니다.
