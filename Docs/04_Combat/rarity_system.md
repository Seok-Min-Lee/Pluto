# 💎 은혜 희귀도 및 성장 시스템 (Rarity System)

이 문서는 `Pluto` 프로젝트의 은혜 희귀도(Rarity) 결정 알고리즘과 등급별 성능 차이를 정의합니다.

---

## 1. 희귀도 등급 (Rarity Tiers)

| 등급 | 색상(Hex) | 성능 배율 (Base 대비) | 설명 |
| :--- | :--- | :--- | :--- |
| **Common** | #FFFFFF | 1.0x | 가장 기본적인 등급. |
| **Rare** | #4A90E2 | 1.3x ~ 1.5x | 보강된 수치를 가짐. |
| **Epic** | #9013FE | 1.8x ~ 2.0x | 강력한 성능 향상. |
| **Heroic** | #F5A623 | 2.3x ~ 2.5x | 전용 교체 은혜로만 획득 가능. |
| **Legendary** | #F8E71C | Special | 고유의 시그니처 로직 추가. |

---

## 2. 희귀도 결정 알고리즘 (Sequential Roll)

은혜가 생성될 때, 시스템은 가장 높은 등급부터 순차적으로 확률을 굴립니다.

1.  **Step 1: Legendary/Duo Check** (조건 충족 시 3~10%)
2.  **Step 2: Epic Check** (기본 5~15%)
3.  **Step 3: Rare Check** (기본 20~40%)
4.  **Step 4: Common Default** (위의 모든 체크 실패 시)

> [!TIP]
> 상위 등급의 체크가 실패하면 바로 다음 하위 등급의 확률을 체크하므로, 플레이어의 운(Luck) 스탯은 Step 1~3의 모든 확률에 보너스를 줍니다.

---

## 3. 등급별 성장 공식 (80-60-40)

모든 은혜는 레벨업 시 **Common 등급의 기초 수치(Base Value)**를 기준으로 성장합니다.

- **LV 2:** Base + (Base * 0.8)
- **LV 3:** LV 2 + (Base * 0.6)
- **LV 4:** LV 3 + (Base * 0.4)
- **LV 5+:** 이전 레벨 + (Base * 0.3)
