# 🦠 상태 이상(Status Effect) 데이터베이스

이 문서는 게임 내 모든 디버프 및 강화 효과의 로직과 수치를 정의합니다. `boon_list.md` 최신 기안을 기반으로 작성되었습니다.

---

## 1. 개요 (Overview)

상태 이상은 `StatusInstance`에 의해 관리되며, 중첩(Stack) 및 지속 시간(Duration) 매커니즘을 가집니다.

---

| ID | 이름 | 효과 설명 | 주요 지표 | 연결 신 |
| :--- | :--- | :--- | :--- | :--- |
| `Stat_Jolted` | **충격(Jolted)** | 적이 공격 시 본인 및 주변에 벼락 피해. | 10초 지속 | 제우스 |

---

## 3. 공통 매커니즘

1. **상태 이상 갱신:** 동일한 상태 이상이 다시 부여되면 원칙적으로 **지속 시간이 초기화**됩니다.
2. **특수 중첩:** 숙취(Hangover)와 같이 최대 중첩수가 정해진 경우, 최대치 도달 시 지속 시간만 갱신됩니다.
3. **상태 이상 해제:** `StatusInstance.Clear()` 호출 시 모든 이펙트와 로직이 중단됩니다.
4. **특혜 상성(Privileged Status):** 서로 다른 상태 이상이 2개 이상 활성화된 적에게는 최종 피해량이 증폭됩니다. (밤의 거울 사양)
